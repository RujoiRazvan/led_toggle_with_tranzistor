# Led toggle by button using a transistor
## My first project with a transistor

Components
* PushButton
* Green and Orange Led - 5mm
* 2x 1kOhm Resistor
* 1x 10kOhm Resistor
* BC337 - NPN Transistor
* Jumper cables
